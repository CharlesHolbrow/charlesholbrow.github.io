Three ways to think about quadratic equations

Feb 15, 2021

Standard form

\[\begin{align} f(x) &= ax^2 + bx + c\\ f(x) &= x^2 + b'x + c'\\ \end{align}\]

In terms of its roots

If we know the roots, \(r\), and \(s\) for a “normalized” function, \(f\)

\[\begin{align} f(r) &= 0\\ f(s) &= 0\\ f(x) &= (x-r)(x-s) && \text{A quadratic function defined in terms of it roots}\\ f(x) &= x^2 -(r+s)x + rs && \text{expands to the form that looks like this}\\ \end{align}\]

From here, we can see that \[\begin{align} b' &= -(r+s)\\ c' &= rs\\ \end{align}\]

The parabola mid-point and distance

Say we have a parabola in the form

\[\begin{equation} ax^2 + bx + c \end{equation}\]

Remember that \(b'\) is the negative sum of of the two roots, \(r\) and \(s\). That means that we can get the mid-point, \(m\) with the following equation (\(b'\) is the normalized value \(b/a\)): \[\begin{equation} m = -\frac{b'}{2} \end{equation}\]

To get the \(\pm\) distance of the of the roots from the mid point, remember that \(c'\) is a product of \(r\) and \(s\). This product can be expressed as a difference of squares.

\[\begin{align} c' &= rs\\ c' &= (m - d)(m + d)\\ c' &= m^2 - d^2\\ d^2 &= m^2 - c'\\ d &= \sqrt{m^2 - c'}\\ \end{align}\]

…where \(d\) is the distance of the roots from the mid-point, \(m\). From which we can deduce:

\[\begin{equation} r, s = m \pm \sqrt{m^2 - c'} \end{equation}\]

  • The key is really recognizing that \(c'\) is a product.
  • You can think about products as
    • A mean of the two factors (the mid-point) and a (\(\pm\)) distance from that mean…
    • …which can be expressed as a difference of squares
import matplotlib
import matplotlib.pyplot as plt
import numpy as np

# Data for plotting
x = np.arange(-1.0, 8.0, 0.01)
y = x**2 -(7*x) + 6

fig, ax = plt.subplots()
ax.plot(x, y)

ax.set(xlabel='x axis', ylabel='y axis', title='a quadratic curve')

ax.grid()
ax.grid()

print(np.min(y))

ax.set_aspect('equal')
ax.grid(True, which='both')

# set the x-spine (see below for more info on `set_position`)
ax.spines['left'].set_position('zero')

# turn off the right spine/ticks
ax.spines['right'].set_color('none')
ax.yaxis.tick_left()

# set the y-spine
ax.spines['bottom'].set_position('zero')

# turn off the top spine/ticks
ax.spines['top'].set_color('none')
ax.xaxis.tick_bottom()

plt.show()
